1. Why CSS selectors are used in HTML? answer- > CSS selectors are used in HTML to target and style specific elements on a web page. HTML provides the structure and content of a webpage, while CSS is used to define the presentation and styling of those HTML elements. Selectors in CSS allow you to specify which HTML elements should be affected by the styles you define. This separation of structure (HTML) and presentation (CSS) is a fundamental principle of web design, as it allows for greater flexibility, maintainability, and consistency in the appearance of a website. 2. What are the different types of Selectors in CSS? answer-> There are several types of CSS selectors: Type Selectors: These target elements based on their tag name. For example, p would target all

elements. Class Selectors: These target elements with a specific class attribute. They are denoted by a period followed by the class name, such as .example-class. ID Selectors: These target a specific element with a unique ID attribute. They are denoted by a hash symbol followed by the ID name, like #example-id. How do you include external fonts and apply them? answer-> To include external fonts and apply them to your website, you can follow these steps: Choose a Font: Select the font you want to use from a font provider like Google Fonts. Link the Font: In the section of your HTML document, include a tag to link the font stylesheet. This tag will be provided by the font provider or can be set up for self-hosting. Apply the Font: In your CSS, use the font-family property to specify the font for specific elements. For example: body { font-family: 'Your Chosen Font', sans-serif; } Fallback Fonts: It's a good practice to include a fallback font family (e.g., sans-serif or serif) in case the chosen font fails to load.
